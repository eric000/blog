# 模板字符串

## 用法

- `123456`，反引号包裹，包含反引号这加 \\`
- 空格、缩进、换行 会保留
- ${} 之中 嵌入表达式or变量，可以再嵌套${}

## 标签模板

```js
function fn(tmpl, ...values) {
  console.log(tmpl, values) 
  // ["Hello I'm ", ", ", " year old", raw: Array(3)]  ["tom", 20]
  // raw 一个模板字符串的原始字符串的 不会转义相关
}

let Tom = {
  name: 'tom',
  age: 20
}

fn`Hello I'm ${Tom.name}, ${Tom.age} year old`
```
